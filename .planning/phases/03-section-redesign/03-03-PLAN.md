# Phase 3.3: FinalCTA, Footer & Final Polish — Plan

## Objective

Complete section redesign with FinalCTA and Footer, then perform final responsive and consistency polish across all sections.

## Prerequisites

- Phase 2 complete (design tokens established)
- Plans 03-01 and 03-02 complete
- All sections using design system utilities

## Design Direction

### FinalCTA: High-Contrast Action Focus
- Remove background animation/glow
- Maximum contrast CTA button
- Minimal supporting text
- Phone number as primary action

### Footer: Utilitarian Links
- Information-dense but clean
- Reduce link categories (many don't exist yet)
- Copyright and essential links only
- Consider removing decorative background

## Tasks

### Task 1: Redesign FinalCTA Section

**Goal**: High-contrast, single-focus call to action

**Current issues**:
- Animated background glow (decorative)
- Dual CTA buttons
- DOM manipulation hover handlers
- Inline styles

**File**: `src/components/FinalCTA.jsx`

**Target structure**:
```jsx
export default function FinalCTA() {
  return (
    <section id="book" className="py-32 bg-void border-t border-white/5">
      <div className="max-w-4xl mx-auto px-6 lg:px-12 text-center">
        {/* Headline */}
        <h2 className="heading-section mb-6">
          Ready to Install an <span className="text-brand-blue">AI System</span>?
        </h2>

        {/* Single line CTA prompt */}
        <p className="body-lg text-slate-400 mb-10">
          Call now for a fit check conversation
        </p>

        {/* Primary CTA - maximum contrast */}
        <a
          href={`tel:${CONTACT.phone}`}
          className="inline-flex items-center gap-3 px-12 py-6 bg-white text-void font-bold text-xl hover:bg-brand-blue hover:text-white transition-colors"
        >
          {CONTACT.phoneDisplay}
        </a>

        {/* Secondary option */}
        <p className="mt-8 text-sm text-slate-500">
          or <a href="mailto:hello@stratablue.ai" className="text-brand-blue hover:text-white transition-colors">send an email</a>
        </p>
      </div>
    </section>
  );
}
```

**Changes**:
1. Remove animated background glow div
2. Remove dual button layout
3. Use white button for maximum contrast
4. Remove all inline styles and DOM handlers
5. Simplify text content

**Verification**: CTA is unmissable, loads fast
